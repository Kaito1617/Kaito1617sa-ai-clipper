# 🔊 SFX — Effets Sonores

Ce dossier contient les effets sonores (SFX) utilisés automatiquement par AI Clipper lors du montage.

## Nommage des fichiers

Les fichiers sont choisis automatiquement selon leur nom. Utilise ces préfixes :

| Préfixe | Utilisation | Mode |
|---------|-------------|------|
| `whoosh_*.mp3` | Transitions rapides | Tous |
| `impact_*.mp3` | Moments d'impact | Football, Anime |
| `boom_*.mp3` | Moments épiques | Anime |
| `vine_boom_*.mp3` | Réactions drôles | Streamer |
| `bruh_*.mp3` | Moments comiques | Streamer |

## Où trouver des SFX libres de droits

### Sites gratuits (licence libre)
- **Freesound.org** — https://freesound.org (CC0 / CC-BY)
- **Pixabay Audio** — https://pixabay.com/sound-effects/
- **Mixkit** — https://mixkit.co/free-sound-effects/
- **ZapSplat** — https://www.zapsplat.com (gratuit avec compte)
- **SoundBible** — https://soundbible.com

### SFX recommandés à télécharger

1. **Whoosh / Swipe** (transition) — cherche "whoosh swipe" sur Freesound
2. **Impact boom** (moments épiques) — cherche "impact hit" sur Freesound
3. **Vine boom** (classique meme) — cherche "vine boom sound effect" sur Pixabay
4. **Bruh sound** — cherche "bruh sound" sur Pixabay

## Format accepté

- **MP3** (recommandé)
- **WAV**
- **OGG**

assets/sfx/
