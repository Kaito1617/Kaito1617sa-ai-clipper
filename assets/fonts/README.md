# 🔤 Fonts — Polices de caractères

Ce dossier contient les polices utilisées pour les sous-titres animés des Shorts.

## Police recommandée : Montserrat Bold

**Montserrat Bold** est la police idéale pour les sous-titres style YouTube Shorts / CapCut.

### Téléchargement

1. Va sur **Google Fonts** :
   https://fonts.google.com/specimen/Montserrat

2. Clique sur **"Download family"**

3. Dans l'archive ZIP, récupère le fichier :
   `Montserrat-Bold.ttf`

4. Copie-le dans ce dossier : `assets/fonts/Montserrat-Bold.ttf`

### Alternative : Impact (style meme)

Pour un style plus "meme" / impactant :
- **Impact** est disponible nativement sur Windows : `C:\Windows\Fonts\impact.ttf`
- Sur Linux : `sudo apt install ttf-mscorefonts-installer`

### Autres polices sympas (libres de droits)

| Police | Style | Lien |
|--------|-------|------|
| **Bebas Neue** | Condensé, impactant | https://fonts.google.com/specimen/Bebas+Neue |
| **Anton** | Bold, moderne | https://fonts.google.com/specimen/Anton |
| **Black Han Sans** | Asiatique/Modern | https://fonts.google.com/specimen/Black+Han+Sans |
| **Boogaloo** | Fun, décontracté | https://fonts.google.com/specimen/Boogaloo |

## Fallback automatique

Si aucune police n'est trouvée dans ce dossier, AI Clipper utilisera automatiquement **DejaVu Sans Bold** (disponible sur Linux/Mac) ou la police système.
