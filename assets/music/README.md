# 🎵 Music — Musiques de fond

Ce dossier contient les musiques de fond optionnelles pour tes Shorts.

## Utilisation

Place tes fichiers musique dans ce dossier. AI Clipper les choisira automatiquement (de façon aléatoire) pour les Shorts générés.

## Nommage recommandé

Pour cibler la musique selon le mode :

| Nom du fichier | Mode associé |
|----------------|-------------|
| `football_*.mp3` | Mode Football |
| `anime_*.mp3` | Mode Anime |
| `streamer_*.mp3` | Mode Streamer |

Sans préfixe spécifique, la musique peut être utilisée dans tous les modes.

## Volume

Le volume de la musique de fond est automatiquement réduit (environ 10-15% du volume original) pour ne pas couvrir l'audio principal de la vidéo.

Tu peux ajuster ce réglage dans les fichiers `modes/*.json` (paramètre `volume_musique_fond`).

## Sources de musique libre de droits

⚠️ **Attention** : n'utilise pas de musique protégée par copyright sur YouTube !

### Sites de musique libre (royalty-free)
- **YouTube Audio Library** — https://studio.youtube.com/channel/music
- **Pixabay Music** — https://pixabay.com/music/
- **Free Music Archive** — https://freemusicarchive.org
- **ccMixter** — https://ccmixter.org
- **Incompetech** (Kevin MacLeod) — https://incompetech.com
- **Bensound** — https://www.bensound.com

### Par style / mode

**⚽ Football :** Musique épique, orchestrale, électro énergique
→ Cherche "epic sport background music" sur Pixabay

**🎌 Anime :** OST style anime, lo-fi épique, synthwave japonais
→ Cherche "anime background music" sur YouTube Audio Library

**🎮 Streamer :** Trap, lo-fi hiphop, phonk, électro gaming
→ Cherche "phonk gaming music" ou "trap beat no copyright"

## Formats acceptés

- **MP3** (recommandé)
- **WAV**
- **OGG**
- **M4A**

## Note

Si ce dossier est vide, les Shorts seront générés sans musique de fond (seulement l'audio original de la vidéo).
